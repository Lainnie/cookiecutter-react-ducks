## React Ducks CookieCutter

This cookiecutter comes armed with react, redux, redux-saga, jest and his send by the ducks army.

## Usage

`cookiecutter gh:lainnie/cookiecutter-react-ducks`

`cd my_project`

`git init`

`yarn`

`yarn run test`

`yarn run start`
